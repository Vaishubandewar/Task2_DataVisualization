# Task 2: Data Visualization & Storytelling

## Objective
This project analyzes the Superstore dataset using Power BI and presents insights through data visualization and storytelling.

## Files in This Folder
- Superstore.csv  
- Task2_Report.pdf  
- README.md  

## Visualizations Created
1. Sales by Category  
2. Sales Trend Over Time  
3. Profit vs Sales  
4. Top 10 Customers  
5. Region-wise Sales  

## Insights
- Technology category has the highest sales; Furniture has the lowest.  
- Sales peak every year during November–December.  
- Some items like Tables have high sales but very low or negative profit.  
- Top 10 customers contribute a large portion of total revenue.  
- West region performs best, while South region needs improvement.

## Recommendations
1. Fix pricing/discount issues in low-profit categories like Tables.  
2. Improve promotions for Furniture category.  
3. Prepare higher inventory for Nov–Dec sale season.  
4. Increase marketing focus in South region.  
5. Provide loyalty offers to top customers.

## Tool Used
Power BI Desktop
